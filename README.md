# HTML Resume Project

This is a simple static resume website built using HTML.

## How to Run the Project

### Method 1: Open directly in browser
1. Download or clone the repository.
2. Navigate to the project folder.
3. Double-click the `resume.html` file  
   **OR**
4. Right-click the file → Open with → Any browser (Chrome, Edge, Firefox).

### Method 2: Using VS Code Live Server (Recommended)
1. Open the project folder in VS Code.
2. Install the **Live Server** extension.
3. Right-click `resume.html`.
4. Click **"Open with Live Server"**.

The project will open in your browser at `http://127.0.0.1:5500/`.
